ssh-keygen ca

This article is written to help you understand the process of signing a user certificate with a CA certificate.


# ca 인증서 생성
$ ssh-keygen -t ed25519 -f id_ca

# Generating public/private ed25519 key pair.
# Enter passphrase (empty for no passphrase):
# Enter same passphrase again:
# Your identification has been saved in id_ca
# Your public key has been saved in id_ca.pub
# The key fingerprint is:
# SHA256:WSl1xC/eT7i1xSLLxX/0+xrMrBYJNo9NCLMtkHIt24s demian@DESKTOP-K8O1SPU
# The key's randomart image is:
# +--[ED25519 256]--+
# |       o  .oo    |
# |    . = +. o.    |
# |     o =.=o. .   |
# |      . ++* o .  |
# |       .S+ O = o |
# |      E . . B+* *|
# |           . ==O+|
# |            +..o=|
# |           .. .o=|
# +----[SHA256]-----+

# user 인증서 생성

$ ssh-keygen -t ed25519 -f id_user
# Generating public/private ed25519 key pair.
# Enter passphrase (empty for no passphrase):
# Enter same passphrase again:
# Your identification has been saved in id_user
# Your public key has been saved in id_user.pub
# The key fingerprint is:
# SHA256:rQOxhlgRX96exeZPVsC5ToyfnyzyLVAwgpAIRa8TWTM demian@DESKTOP-K8O1SPU
# The key's randomart image is:
# +--[ED25519 256]--+
# | .+o+Eo o    ... |
# |   .++o+ o +  o. |
# |   o..o . o *o ..|
# |   oo. o o =..+. |
# |  .o. + S + o+o. |
# |    .. . . . ++  |
# |        o   . o..|
# |         . . o.o.|
# |            o.o. |
# +----[SHA256]-----+

# CA key(id_ca) 파일과 id_user.pub파일로 id_user-cert.pub 파일 생성
# -s 공개키에 사인할 CA key 파일 경로
# -I는 certificate_identity 를 지정하며 관리 등을 위한 메타데이터
# -V는 인증서 유효기간을 52주(1년)로 지정
# -z는 생성될 cert.pub 파일에 포함될 serial_number를 지정한다. 이 번호는 cert.pub파일에 포함된 같은 CA의 인증서와 구별하기위해 사용됩니다.


$ ssh-keygen -s id_ca -I lee -n lee,admin -V +52w -z 12345 id_user.pub
Signed user key id_user-cert.pub: id "lee" serial 12345 for lee,admin valid from 2023-08-09T09:47:00 to 2024-08-07T09:48:32

# -f 는 key file을 지정한다.
# -L은 cert.pub 파일에 있는 인증서들의 목록을 출력하는 옵션이다.
$ ssh-keygen -L -f id_user-cert.pub
id_user-cert.pub:

        Key ID: "lee"
        Serial: 12345
        Valid: from 2023-08-09T09:47:00 to 2024-08-07T09:48:32
        Principals:
                lee
